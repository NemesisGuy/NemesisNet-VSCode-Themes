# NemesisNet VSCode Theme - Quick Start

## ðŸš€ Installation (2 Minutes)

### Windows Quick Install

1. **Open File Explorer** and navigate to:
   ```
   %USERPROFILE%\.vscode\extensions\
   ```
   Or paste this in the address bar: `%USERPROFILE%\.vscode\extensions\`

2. **Copy the entire folder** `NemesisNet-VSCode-Theme` here

3. **Restart VSCode**

4. **Activate theme**:
   - Press `Ctrl+K Ctrl+T`
   - Select any NemesisNet theme

### macOS/Linux Quick Install

1. **Open Terminal** and run:
   ```bash
   cp -r NemesisNet-VSCode-Theme ~/.vscode/extensions/
   ```

2. **Restart VSCode**

3. **Activate theme**:
   - Press `Cmd+K Cmd+T` (macOS) or `Ctrl+K Ctrl+T` (Linux)
   - Select any NemesisNet theme
