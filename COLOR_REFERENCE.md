## Terminal Colors

Each theme includes custom ANSI colors for the integrated terminal:

### Standard ANSI Colors
- **Red**: Errors, git deletions
- **Green**: Success, git additions  
- **Yellow**: Warnings, modified files
- **Blue**: Information, links
- **Magenta**: Special items, decorations
- **Cyan**: Highlights, navigation
- **White/Black**: Text and backgrounds
