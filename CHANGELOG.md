# Changelog

All notable changes to the NemesisNet VSCode Theme will be documented in this file.

## [1.0.0] - 2025-11-20

### Added
- Initial release of NemesisNet Theme extension
- **Base Dark** theme with nemesis blue accents
- **Base Light** theme with clean, professional aesthetics
- **Nemesis Dark** theme with cyberpunk cyan/magenta palette
- **Nemesis Light** theme with neon-on-porcelain aesthetic
- **Aurora Dark** theme with violet/aqua glowing accents
- **Aurora Light** theme with glassy, editorial design
- Semantic highlighting support for all themes
- Comprehensive terminal color support
- Git decoration colors
- Markdown syntax highlighting
- JSON and web development optimizations

### Design Philosophy
- All themes based on official NemesisNet brand guidelines
- Consistent color theory across all variants
- Accessibility-focused contrast ratios
- Premium aesthetic with attention to detail
